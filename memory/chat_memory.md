## 四、Excel 使用注意

- 筛选后只对**可见行**统计：用 **SUBTOTAL**，不用 SUM/AVERAGE。  
  - 求和：SUBTOTAL(9, 列) 或 (109, 列)  
  - 平均：SUBTOTAL(101, 列)  
  - 最大值：SUBTOTAL(104, 列)  
- 排序时选「**扩展选定区域**」，整行一起排，避免列错位。

## 五、测试指标（评估智能体）

- **准确性**：回复对错，1～5 分专家均分，目标均≥4.5。  
- **一致性**：同一问题多次运行输出一致，20 次零差异率目标 100%。  
- **无幻觉性**：答案中事实可追溯、无编造，无幻觉率目标 100%。
